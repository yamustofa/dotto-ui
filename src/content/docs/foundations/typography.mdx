Two typefaces carry the whole system:

- **Display** — `Silkscreen` (`.font-display`), reserved for stacked, uppercase
  display moments.
- **Body & labels** — `Space Mono` (the default `font-mono` / `font-sans`), for
  body text, labels, captions, and controls.

## Usage

```tsx
<h1 className="font-display text-6xl uppercase">Pixel Studio</h1>
<p className="font-mono text-base">Body copy in Space Mono.</p>
```
