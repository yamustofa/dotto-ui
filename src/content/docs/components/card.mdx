## Anatomy

| Part | Role |
| --- | --- |
| `Card` | The bordered surface wrapper |
| `CardHeader` | Title + description stack |
| `CardTitle` | Uppercase mono heading |
| `CardDescription` | Muted supporting text |
| `CardContent` | Main body |
| `CardFooter` | Actions row |
